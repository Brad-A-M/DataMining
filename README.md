# DataMining

kmeans.py returns a dataframe in the following form when `generateClusters(max_iterations)` is called.

| Attribute 1 |  Attribute 2 | ... | class   | cluster   |
| ----------- | ------------ | --- | ------- | --------- |
| value 1     | value 1      | ... | class # | cluster # |
| value 2     | value 2      | ... | class # | cluster # |
| ...         | ...          | ... | ...     | ...       |
